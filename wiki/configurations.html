Configurations:

To simplify the use of the project several aspects of the server and client are configurable.


Server configurations:
All the server configurations are stored as ini files in the Config folder.
You have the ability to setup account, database, logging, networking, player and world settings.


Client configurations:
All the client configurations are stored in the Assets\Scripts\Config folder.
You have the ability to setup encyption, network and client version settings.


"It is recommended to create configurations for everything."
